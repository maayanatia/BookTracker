# 📚 Matriculation Elevation

Matriculation Elevation is an Android application designed to help
Israeli high school students prepare for their **בגרות (Matriculation)
exams** in a more interactive and engaging way.

The app combines **study materials, exercises, progress tracking, and
multimedia features** to make exam preparation easier and more
motivating.

------------------------------------------------------------------------

## 🚀 Features

-   📖 **Theory Lessons**\
    Clear explanations of study material for different matriculation
    subjects.

-   📝 **Practice Exercises**\
    Interactive questions to help reinforce what you learned.

-   📊 **Progress Tracking**\
    Users can track their learning progress and see improvement over
    time.

-   🎵 **Music Features**\
    Optional music support to create a better study atmosphere.

-   👤 **User Accounts**\
    Students can create accounts and store their personal progress.

-   📚 **Bookshelf System**\
    Users can organize learning materials and resources.
