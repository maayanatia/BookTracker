# 📚 BookTracker

BookTracker is an Android application that allows users to search for
books, view detailed information about them, and organize them into
personal bookshelves.

The app integrates the Google Books API to retrieve real-world book data
and uses Firebase services to manage user accounts and store bookshelf
information in the cloud.

------------------------------------------------------------------------

## 🚀 Features

-   🔎 **Search Books**\
    Search for books using the Google Books API.

-   📖 **Book Details**\
    View information such as title, authors, description, and cover
    image.

-   📚 **Personal Bookshelves**\
    Save books into personal shelves for easy organization.

-   👤 **User Accounts**\
    Secure login and registration using Firebase Authentication.

-   ☁️ **Cloud Storage**\
    User bookshelves and data are stored using Firebase Firestore.

-   🖼 **Book Covers**\
    Book images are loaded dynamically using image loading libraries.
